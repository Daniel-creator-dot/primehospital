5. Click your preferred format

6. File downloads with filename like:
   - patients_all_20251111_162358.xlsx
   - patients_legacy_20251111_162400.pdf
   - patients_new_20251111_162402.csv

══════════════════════════════════════════════════════════════════════════════
                        FORMAT COMPARISON
══════════════════════════════════════════════════════════════════════════════

📗 EXCEL (.xlsx):
  ✓ Best for: Data analysis, reporting
  ✓ Features: Colors, formatting, auto-width
  ✓ Size: Medium
  ✓ Speed: Medium
  ✓ Can edit in: Excel, Google Sheets, LibreOffice

📕 PDF:
  ✓ Best for: Printing, sharing, official docs
  ✓ Features: Professional layout, headers
  ✓ Size: Large
  ✓ Speed: Slower (for large datasets)
  ✓ Can view in: Any PDF reader

📄 CSV:
  ✓ Best for: Importing, data transfer
  ✓ Features: Simple, universal format
  ✓ Size: Smallest
  ✓ Speed: Fastest
  ✓ Can open in: Excel, any text editor
