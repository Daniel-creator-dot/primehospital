## 🚀 HOW TO USE

### **Step 1: Access World-Class Dashboard**
```
http://127.0.0.1:8000/hms/hr/worldclass/
```

**You'll see:**
- 8 colorful statistics cards
- Contract expiry alerts (if any)
- Recent leave requests
- Quick action buttons to calendars

---

### **Step 2: Setup Public Holidays**

**Go to:** Admin → Public Holidays → Add

**Example:**
- Holiday Name: Christmas Day
- Date: December 25, 2025
- Recurring: Yes
- Description: Public holiday

**Click:** Save

✅ **Now this holiday will be respected in attendance!**

---

### **Step 3: Link Staff to Contracts**

**Go to:** Admin → Staff Employment Contracts → Add

**Fill in:**
- Staff: Select employee
- Contract: Select from contracts list
- Is Current: ✓
- Monthly Salary: 5000.00
- Annual Salary: 60000.00
- Health Insurance: ✓
- Pension Included: ✓
- Housing Allowance: 500.00
- Transport Allowance: 300.00
- Annual Leave Days: 21
- Sick Leave Days: 10
- Notice Period Days: 30

**Click:** Save

✅ **Staff contract now tracked with automated expiry alerts!**

---

### **Step 4: Track Daily Attendance**

**Go to:** Admin → Attendance Calendar → Add

**Fill in:**
- Staff: Jane Smith
- Attendance Date: Today
- Status: Present
- Check In Time: 08:00
- Check Out Time: 17:00

**Click:** Save

✅ **Hours automatically calculated! (9 hours worked, 1 hour overtime)**

---

### **Step 5: Set Performance Goals**

**Go to:** Admin → Staff Performance Goals → Add

**Fill in:**
- Staff: John Doe
- Goal Title: Improve patient satisfaction scores
- Description: Achieve 95% patient satisfaction in Q4
- Set Date: Today
- Target Date: December 31, 2025
- Status: In Progress
- Progress Percentage: 40
- Target Value: 95%
- Set By: You

**Click:** Save

✅ **Goal tracked with progress monitoring!**

---
