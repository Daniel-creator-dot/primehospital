========================================
SIMPLE AUTO-START SETUP GUIDE
========================================

Make your HMS server start automatically on every boot!

========================================
METHOD 1: Docker Desktop Settings (Easiest)
========================================

1. Open Docker Desktop
2. Click Settings (gear icon, top right)
3. Go to "General" tab
4. Check "Start Docker Desktop when you log in"
5. Click "Apply & Restart"

That's it! Docker Desktop will start automatically.

Then HMS services will start automatically via the startup script.

========================================
METHOD 2: Use the Script
========================================

Run: ENABLE_DOCKER_AUTOSTART.bat

This will open Docker Desktop for you to enable auto-start.

========================================
HOW IT WORKS:
========================================

1. Windows boots
2. Docker Desktop starts automatically (if enabled in settings)
3. HMS startup script runs (from Startup folder)
4. Script waits for Docker Desktop to be ready
5. HMS services start automatically
6. Server is ready at http://192.168.0.102:8000

Total time: 2-3 minutes after boot

========================================
VERIFY IT'S WORKING:
========================================

After enabling auto-start:

1. Restart your computer
2. Wait 2-3 minutes
3. Check services: docker-compose ps
4. Access: http://192.168.0.102:8000

If services are running, auto-start is working! ✅

========================================
TROUBLESHOOTING:
========================================

Docker Desktop not starting automatically?
→ Make sure "Start Docker Desktop when you log in" is checked
→ Check Docker Desktop settings -> General

HMS services not starting?
→ Check Startup folder has HMS_AutoStart.lnk
→ Run startup_hms.bat manually to test
→ Check startup_log.txt for errors

Services start but can't access?
→ Run allow_port_8000_firewall.bat as Administrator
→ Check: docker-compose ps

========================================
MANUAL START (if auto-start fails):
========================================

Just run: START_EVERYTHING.bat

This will:
- Check Docker Desktop
- Start all services
- Configure firewall

========================================
DISABLE AUTO-START:
========================================

To disable:
1. Docker Desktop -> Settings -> General
2. Uncheck "Start Docker Desktop when you log in"
3. Delete HMS_AutoStart.lnk from Startup folder
