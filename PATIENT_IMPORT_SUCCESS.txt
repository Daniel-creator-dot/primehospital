                      QUERY EXAMPLES

1. Find patient by name:
   SELECT * FROM patient_data WHERE fname='Desmond' AND lname='Allotey';

2. Get all female patients:
   SELECT id, fname, lname, DOB FROM patient_data WHERE sex='Female' LIMIT 20;

3. Patients with phone numbers:
   SELECT fname, lname, phone_cell 
   FROM patient_data 
   WHERE phone_cell != '' AND phone_cell != 'NULL'
   LIMIT 20;

4. Recently registered:

   FROM patient_data 
   ORDER BY date DESC 
   LIMIT 10;

5. Count by price level (insurance tier):
