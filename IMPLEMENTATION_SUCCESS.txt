║  TROUBLESHOOTING                                                             ║
╚══════════════════════════════════════════════════════════════════════════════╝

COMMON ISSUES:

   Problem: "Table already exists"
   Solution: Use --skip-drop flag

   Problem: "Module not found"
   Solution: pip install -r requirements.txt

   Problem: "Permission denied"
   Solution: Close database programs, run as admin

   Problem: Import is slow
   Solution: Import by category instead of all at once

   Problem: SQL syntax error
   Solution: Check logs, report specific table

NEED HELP?
   1. Run: python check_import_prerequisites.py
   2. Read: DATABASE_IMPORT_GUIDE.md (troubleshooting section)
   3. Check error messages carefully
   4. Try importing 1-2 tables first
