🚀 INSTALLATION (5 minutes)

Step 1: Install Required Libraries
───────────────────────────────────
pip install deepface opencv-python tensorflow

Step 2: Run Migrations
──────────────────────
python manage.py makemigrations
python manage.py migrate

Step 3: Initialize System
──────────────────────────
python manage.py init_biometric_system

Step 4: Restart Server
──────────────────────
python manage.py runserver


📱 KEY URLs TO ACCESS

1. FRONT DESK LOGIN (Public - No Auth Required)
   http://localhost:8000/hms/biometric/login/
   → Staff can scan face to login and clock in

2. ENROLLMENT INTERFACE (HR/Admin Only)
   http://localhost:8000/hms/biometric/enrollment/
   → Enroll staff with face recognition

3. BIOMETRIC DASHBOARD (Admin Only)
   http://localhost:8000/hms/biometric/dashboard/
   → Monitor authentication activity

4. MY BIOMETRIC PROFILE (Staff Portal)
   http://localhost:8000/hms/biometric/my-profile/
   → Staff view their biometric data

5. ADMIN PANEL
   http://localhost:8000/admin/
   → Navigate to "Biometric System Settings" to configure


🎯 QUICK WORKFLOW

FOR HR/ADMIN - Enrolling Staff:
1. Go to: http://localhost:8000/hms/biometric/enrollment/
2. Select staff member from list
3. Click "Start Camera"
4. Have staff look at camera
5. Click "Capture & Enroll"
6. Done! Staff is now enrolled.

FOR STAFF - Daily Login:
1. Go to: http://localhost:8000/hms/biometric/login/
2. Look at camera
3. Click "Authenticate with Face ID"
4. Automatically logged in + attendance marked
5. Redirected to staff dashboard


⚙️ KEY SETTINGS (Django Admin)

Go to: Admin → Biometric System Settings

Essential Settings:
• System Enabled: ON
• Allow Password Fallback: ON (recommended)
• Enable Liveness Detection: ON (security)
• Min Confidence Score: 85.00 (default)
• Auto Create Attendance: ON (convenience)

Security Settings:
• Max Failed Attempts: 3
• Lockout Duration: 15 minutes
• Enable Anti-Spoofing: ON


🔧 TROUBLESHOOTING

Camera Not Working?
→ Check browser permissions
→ Must use HTTPS (or localhost)
→ Refresh page and allow camera

Enrollment Fails - Low Quality?
→ Improve lighting
→ Clean camera lens
→ Move closer to camera
→ Remove glasses

Authentication Fails - No Match?
→ Check if staff is enrolled
→ Verify biometric is not locked
→ Check admin panel for status
→ Re-enroll if needed

Slow Performance?
→ Install tensorflow-gpu for speed
→ Use lighter model (VGG-Face)
→ Enable caching in settings
