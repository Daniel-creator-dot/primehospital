======================================================================
                        📋 WHAT IT DOES
======================================================================

1. ✅ Imports Table Structures
   - Creates all 4 tables in PostgreSQL
   - Converts MySQL syntax to PostgreSQL

2. ✅ Imports Data (if INSERT statements exist)
   - Reads INSERT statements from SQL files
   - Imports all rows into tables

3. ✅ Extracts Patient IDs
   - Finds all unique PIDs from:
     * patient_access_offsite
     * patient_access_onsite
     * patient_reminders

4. ✅ Creates Patient Records
   - Creates Patient for each unique PID
   - MRN format: PMC-LEG-{PID}
   - Example: PID 123 → MRN: PMC-LEG-000123

5. ✅ Generates QR Codes
   - Automatically creates QR code for each patient
   - QR codes are ready to use immediately!


======================================================================
                        🎯 PATIENT MRN FORMAT
======================================================================

Patients are created with MRNs:
  PMC-LEG-{PID}

Examples:
  PID 1     → MRN: PMC-LEG-000001
  PID 123   → MRN: PMC-LEG-000123
  PID 4567  → MRN: PMC-LEG-004567


======================================================================
                        📱 QR CODE GENERATION
======================================================================

QR codes are AUTOMATICALLY generated via Django signals!

Each patient gets:
  ✅ Unique QR token
  ✅ QR code image (PNG)
  ✅ QR code data (patient UUID)

View QR codes:
  1. Go to: http://localhost:8000/hms/patients/
  2. Click on any patient
  3. Click "QR Card" button
  4. Download or print!
