## 🚨 Common Issues & Quick Fixes

### Issue: Build Fails

**Fix:**
```bash
git update-index --chmod=+x build.sh
git commit -m "Fix build script permissions"
git push
```

### Issue: Static Files Not Loading

**Fix:** Check these in `settings.py`:
```python
STATIC_ROOT = BASE_DIR / 'staticfiles'
STATICFILES_STORAGE = 'whitenoise.storage.CompressedManifestStaticFilesStorage'
```

### Issue: Database Connection Error

**Fix:** 
1. Verify `DATABASE_URL` environment variable is set
2. Wait 2-3 minutes for database to initialize
3. Check PostgreSQL service is "Live" (green status)

### Issue: Email Not Sending

**Fix:**
1. Use Gmail App Password (not regular password)
2. Verify `EMAIL_USE_TLS=True` and `EMAIL_PORT=587`
3. Test with: `python manage.py sendtestemail test@example.com`

### Issue: Service Keeps Sleeping (Free Tier)

**Fix:**
1. Upgrade to Starter plan ($7/month)
2. Or use UptimeRobot to ping every 10 minutes
