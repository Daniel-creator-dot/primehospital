Options:
   --skip-drop                   - Keep existing tables
   --dry-run                     - Preview only
   --tables blood_donors         - Specific tables
   --detailed                    - Detailed stats
   --check-integrity             - Data checks


================================================================================
                     TROUBLESHOOTING QUICK FIX
================================================================================

Problem: "Table already exists"
Fix: Use --skip-drop flag

Problem: "Module not found"
Fix: pip install -r requirements.txt

Problem: "Permission denied"
Fix: Close database programs, run as administrator

Problem: Import is slow
Fix: Import specific categories instead of all
