========================================
TO FIX POSTGRESQL (5 MINUTES):
========================================

STEP 1: Open pgAdmin
---------------------
- Search for "pgAdmin" in Start Menu
- Open pgAdmin 4 (or your PostgreSQL Desktop app)

STEP 2: Connect to PostgreSQL
-------------------------------
- In pgAdmin, you'll see your PostgreSQL server
- Click on it (you may need to enter password)
- If you don't remember the password, you set it during PostgreSQL installation

STEP 3: Set postgres User Password to "postgres"
-------------------------------------------------
1. In pgAdmin left sidebar, expand "Login/Group Roles"
2. Right-click "postgres" → Properties
3. Click "Definition" tab
4. Enter password: postgres
5. Click "Save"

STEP 4: Create Database
------------------------
1. In pgAdmin, right-click "Databases" → Create → Database
2. Name: hms_db
3. Owner: postgres
4. Click "Save"

STEP 5: Run Migrations
-----------------------
Open PowerShell or Command Prompt in this folder and run:

   docker-compose exec web python manage.py migrate

That's it! Your app will be fully working.

========================================
IF YOUR PASSWORD IS DIFFERENT:
========================================

If your PostgreSQL password is NOT "postgres", tell me what it is and I'll update the .env file for you.
