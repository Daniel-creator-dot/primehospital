  3. Download complete!

DELETE OLD:
  1. Click "Delete Old Backups"
  2. Choose age (30 days, 60 days, etc.)
  3. Confirm
  4. Old backups removed

══════════════════════════════════════════════════════════════════════════════
                        📊 BACKUP RECOMMENDATIONS
══════════════════════════════════════════════════════════════════════════════

FREQUENCY:
  Daily:    Create auto backup every day
  Weekly:   Create named backup with "weekly_YYYY_WW"
  Monthly:  Create named backup with "monthly_YYYY_MM"
  Before:   Create backup before any major operation

RETENTION:
  Keep recent:   7 daily backups
  Keep weekly:   4 weekly backups (1 month)
  Keep monthly:  12 monthly backups (1 year)
  Delete old:    Backups older than 30-90 days

STORAGE:
  Local:     C:\Users\user\chm\backups\database\
  External:  Download and store on external drive
  Cloud:     Upload critical backups to cloud storage
  Multiple:  Keep backups in multiple locations
