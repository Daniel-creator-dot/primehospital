======================================================================
                        💡 HOW TO USE
======================================================================

RECORD REVENUE:
  1. Admin → Revenues → Add Revenue
  2. Select category, enter amount, patient
  3. Save
  4. Appears in financial reports

RECORD EXPENSE:
  1. Admin → Expenses → Add Expense
  2. Enter vendor, category, amount
  3. Submit for approval
  4. Create payment voucher
  5. Mark as paid

CREATE JOURNAL ENTRY:
  1. Admin → Advanced Journal Entries → Add
  2. Select journal type, enter date
  3. Add lines (debits and credits must balance!)
  4. Save
  5. Post to general ledger

RUN REPORTS:
  1. Visit: /hms/accounting/
  2. Click any report link
  3. Select date range
  4. View results
  5. Export if needed
