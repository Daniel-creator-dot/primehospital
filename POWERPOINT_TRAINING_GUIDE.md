## 🎨 DESIGN TIPS

### Color Scheme
- **Primary:** Hospital brand colors
- **Secondary:** Professional blues/greens
- **Accent:** Use sparingly for highlights

### Fonts
- **Headings:** Bold, 24-32pt
- **Body:** Regular, 18-20pt
- **Code/URLs:** Monospace, 16pt

### Images
- Use actual screenshots from system
- Crop to show relevant areas
- Add arrows/callouts for important areas
- Keep images clear and readable

### Layout
- One main idea per slide
- Use bullet points (max 5-6 per slide)
- White space is important
- Consistent alignment

## 🎬 PRESENTATION TIPS

### Before Presenting:
1. **Practice:** Run through entire presentation
2. **Test:** Make sure all screenshots are clear
3. **Prepare:** Have system open for live demo
4. **Backup:** Have PDF version ready

### During Presentation:
1. **Pace:** Don't rush, allow time for questions
2. **Engage:** Ask questions to audience
3. **Demo:** Show live system when possible
4. **Pause:** Give time for note-taking

### After Presentation:
1. **Q&A:** Answer all questions
2. **Resources:** Share documentation
3. **Follow-up:** Provide contact for support
4. **Feedback:** Collect feedback forms
