                        🎯 HOW IT WORKS
======================================================================

SMART NOTIFICATIONS:
  ✅ Only shows if user has permission
  ✅ Only shows if there are pending requests
  ✅ Shows exact count
  ✅ Direct link to approval page
  ✅ Dismissible (can close alert)

  - Admin approval: requires 'can_approve_procurement_admin'

REAL-TIME:

  - Shows current pending count
  - No caching
