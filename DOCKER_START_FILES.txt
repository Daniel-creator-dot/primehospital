========================================
DOCKER START FILES - USE ANY OF THESE
========================================

MAIN FILE (RECOMMENDED):
------------------------
START_DOCKER.bat
  - Double-click this file to start everything
  - Checks Docker Desktop
  - Creates .env if needed
  - Builds and starts all services

ALTERNATIVE FILES:
------------------
docker-start.bat
  - Original Docker start script
  - Updated for PostgreSQL Desktop

start-docker.bat
  - Alternative start script

========================================
HOW TO USE:
========================================

1. Make sure PostgreSQL Desktop is running
2. Make sure Docker Desktop is running
3. Double-click: START_DOCKER.bat
4. Wait for services to start
5. Open: http://localhost:8000

========================================
WHAT THESE FILES DO:
========================================

- Check if Docker Desktop is running
- Check/create .env file
- Build Docker images
- Start all services (web, redis, celery, etc.)
- Connect to PostgreSQL Desktop on your machine
