# Quick Start Guide - Tabular Lab Reports

## 🚀 Get Started in 3 Steps

### Step 1: Access the System
Navigate to: **Laboratory Dashboard** → **In Progress Results** → Click **"Enter"** button (table icon)

### Step 2: Enter Results
- Values are entered in the "Value" column
- Units and reference ranges are shown for guidance
- Auto-calculated fields update automatically (highlighted in blue)

### Step 3: Save
- Click **"Save Results"** to save
- Or click **"Save & Mark Complete"** to finish

---

## 📋 Supported Tests

| Test Type | Parameters | Auto-Calculations |
|-----------|------------|-------------------|
| **Full Blood Count (FBC)** | 14 | None |

| **Renal Function Tests (RFT)** | 9 | None |
| **Lipid Profile** | 8 | VLDL, Ratios, Non-HDL |
| **Thyroid Function (TFT)** | 5 | None |
| **Blood Glucose** | 4 | None |
| **Electrolytes** | 7 | None |

## 💡 Pro Tips

### Speed Tips
- Use **Tab** to move between fields quickly
- Press **Ctrl+S** to save instantly
- Auto-save protects your work (uses browser storage)

### Accuracy Tips
- Compare values with reference ranges (shown on the right)
- Auto-calculated fields update in real-time
- Check values before saving

### Workflow Tips
1. Keep analyzer printout handy
2. Enter all values at once
3. Review auto-calculations
4. Add clinical notes if needed
5. Save and move to next result

---

## 🎯 Common Scenarios

### Scenario 1: Entering a Full Blood Count
```
1. Click "Enter" on FBC result
2. System auto-detects FBC
3. Fill in:
   - WBC: 7.2
   - RBC: 4.8
   - HGB: 13.5
   - ... (continue)
4. Fill differential:
   - Neutrophils: 60%
   - Lymphocytes: 30%
   - ... (continue)
5. Add notes if needed
6. Click "Save & Mark Complete"
```

### Scenario 2: Entering Lipid Profile
```
1. Click "Enter" on Lipid result
2. Fill in:
   - Total Cholesterol: 180
   - Triglycerides: 120
   - HDL: 55
   - LDL: 100
3. Watch auto-calculations:
   - VLDL: 24 (auto)
   - Ratios: (auto)
4. Save
```

### Scenario 3: Wrong Test Type?
```
1. Click "Change Test Type" button
2. Select correct type from modal
3. New form loads automatically
4. Enter data
```

## ⚠️ Important Notes

### Reference Ranges
- Shown for guidance only
- M = Male, F = Female
- May vary by lab standards

### Auto-Calculations
- Cannot be edited manually
- Update as you type
- Based on standard formulas

### Data Safety
- Auto-saves to browser
- Don't close tab without saving
- Data clears after successful save

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Test type wrong | Click "Change Test Type" button |
| Auto-calc not working | Fill required fields first |
| Lost data after refresh | Should auto-load from browser storage |
| Can't save | Check internet connection |
| Wrong value entered | Just retype and save again |

| Status | Meaning | When to Use |
|--------|---------|-------------|

| **In Progress** | Being tested | Test running, results not ready |
