# Drug Classification Guide - Access Instructions

## ✅ Setup Complete

The Drug Classification Guide has been successfully added to your system.

## 📍 Where to Find It

### 1. **Pharmacy Dashboard** (Main Location)
- Go to: `/hms/pharmacy/` or click "Pharmacy" in your navigation
- Look for the **blue button** with book icon: **"📚 Drug Classification Guide"**
- Located in the "Quick Access Buttons" section at the top of the pharmacy dashboard

### 2. **Sidebar Navigation** (For Pharmacists & Doctors)
- The link appears in the left sidebar navigation menu
- Title: "Drug Classification Guide"
- Icon: Book icon
- Only visible if you're logged in as:
  - **Pharmacist** role
  - **Doctor** role

### 3. **Direct URL Access**
- URL: `/hms/drug-classification-guide/`
- Full URL: `http://your-server:port/hms/drug-classification-guide/`

## 🔧 Troubleshooting

### If you can't see it:

1. **Refresh your browser** (Ctrl+F5 or Cmd+Shift+R for hard refresh)
2. **Restart Django server** if it's running:
   ```bash
   # Stop the server (Ctrl+C)
   # Then restart:
   python manage.py runserver
   ```
3. **Check your user role**:
   - Make sure you're logged in as a user with "Pharmacist" or "Doctor" role
   - Admin users can also access it via direct URL
4. **Clear browser cache** if needed
5. **Check the pharmacy dashboard** - the button should be visible in the Quick Access section
