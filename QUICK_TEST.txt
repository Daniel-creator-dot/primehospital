QUICK TEST - DUPLICATE PREVENTION
═══════════════════════════════════════════════════════════════

1. Make sure server is running:
   - Check if you see a window with Django server output
   - Or run: python manage.py runserver 0.0.0.0:8000

2. Open browser:
   http://localhost:8000/hms/patients/new/

3. Register patient:
   First Name: Test
   Last Name: Duplicate
   Phone: 0241234567
   Email: test@example.com
   Date of Birth: 1990-01-01
   
   Click "Register Patient"
   ✅ Should succeed

4. Try again with SAME data:
   Go back to registration form
   Fill in EXACT SAME information
   Click "Register Patient"
   
   ❌ Should show error: "Duplicate patient detected!"
   ❌ Should NOT create second patient

5. Check patient list:
   http://localhost:8000/hms/patients/
   Search for "Test Duplicate"
   
   ✅ Should find only ONE patient

═══════════════════════════════════════════════════════════════

IF DUPLICATE IS CREATED = SYSTEM NOT WORKING
IF ERROR SHOWN = SYSTEM WORKING ✅
