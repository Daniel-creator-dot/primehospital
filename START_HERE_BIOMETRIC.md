## 🎯 3-STEP QUICK START

### Step 1: Install (2 minutes)
```bash
pip install deepface opencv-python tensorflow
```

### Step 2: Setup (1 minute)
```bash
python manage.py makemigrations
python manage.py migrate
python manage.py init_biometric_system
```

### Step 3: Start Using (1 minute)
```bash
python manage.py runserver
```

Then open: **http://localhost:8000/hms/biometric/login/**

## 🎬 HOW TO USE

### Enrolling Staff (HR)
1. Go to enrollment page
2. Select staff member
3. Click "Start Camera"
4. Capture face (3-5 seconds)
5. Done! Staff can now login with face

### Staff Login (Front Desk)
1. Go to biometric login page
2. Look at camera
3. Click "Authenticate"
4. Logged in + attendance marked automatically!

## 🔧 QUICK TROUBLESHOOTING

**Camera not working?**
→ Allow camera permissions in browser
→ Use Chrome or Edge
→ Must be HTTPS (or localhost)

**Enrollment fails?**
→ Improve lighting
→ Remove glasses
→ Look directly at camera

**Authentication fails?**
→ Check if staff is enrolled
→ Verify good lighting
→ Contact HR if issues persist
