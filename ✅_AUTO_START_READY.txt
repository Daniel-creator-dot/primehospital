HOW IT WORKS:
========================================

On Every Boot:
1. Windows starts
2. Docker Desktop starts automatically
3. HMS startup script runs (from Startup folder)
4. Script waits for Docker Desktop (up to 2 minutes)
5. Once Docker Desktop is ready, HMS services start
6. Your server is ready at http://192.168.0.102:8000

Total Time: 2-3 minutes after boot
