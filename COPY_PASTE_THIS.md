# ✅ Copy & Paste These Into Render

**Go to: Render Dashboard → hms-web → Environment → Add Environment Variable**

Click "Add Environment Variable" for each line below:

---

## 1. ALLOWED_HOSTS
```
ALLOWED_HOSTS
```
Paste value:
```
your-app-name.onrender.com
```
*(Replace your-app-name with your actual Render service name)*

---

## 2. SITE_URL
```
SITE_URL
```
Paste value:
```
https://your-app-name.onrender.com
```

---

## 3. EMAIL_HOST
```
EMAIL_HOST
```
Paste value:
```
smtp.gmail.com
```

---

## 4. EMAIL_HOST_USER
```
EMAIL_HOST_USER
```
Paste value:
```
youremail@gmail.com
```
*(Your actual Gmail address)*

---

## 5. EMAIL_HOST_PASSWORD
```
EMAIL_HOST_PASSWORD
```
Paste value:
```
your-16-character-app-password
```
*(Get from: https://myaccount.google.com/apppasswords)*

---

## 6. EMAIL_PORT
```
EMAIL_PORT
```
Paste value:
```
587
```

---

## 7. EMAIL_USE_TLS
```
EMAIL_USE_TLS
```
Paste value:
```
True
```

---

## 8. DEFAULT_FROM_EMAIL
```
DEFAULT_FROM_EMAIL
```
Paste value:
```
noreply@yourhospital.com
```

---

## 9. SMS_API_KEY
```
SMS_API_KEY
```
Paste value:
```
your-sms-api-key
```

---

## 10. SMS_SENDER_ID
```
SMS_SENDER_ID
```
Paste value:
```
PrimeCare
```

---

## ✅ After Adding All 10

Click **"Save Changes"** button at bottom.

Wait 2-3 minutes for Render to redeploy.

**Done!** 🎉
