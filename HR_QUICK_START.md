## 📝 **Leave Request Checklist**

Before submitting leave request, make sure:

- [ ] Dates are correct
- [ ] Reason is clear
- [ ] Check leave balance
- [ ] Add covering staff (if required)
- [ ] Upload documents (if needed)
- [ ] Add handover notes
- [ ] Submit at least 7 days in advance

---

## ⚠️ **Important Rules**

### **For Staff:**
1. Submit requests **7 days in advance** (except emergencies)
2. Sick leave **> 3 days requires medical certificate**
3. Ensure **covering staff** is assigned
4. **Cannot cancel** approved leaves (contact HR)
5. **Balance is deducted** only after approval

### **For Managers:**
1. **Review within 48 hours**
2. **Provide clear rejection** reasons
3. **Check team coverage** before approving
4. **Balance business needs** with staff welfare
5. **Document decisions** properly

## 🆘 **Troubleshooting**

### **Problem: Cannot access staff dashboard**

**Solutions:**
- Ensure you're logged in
- Check you have a Staff profile
- Contact admin to create profile

### **Problem: Leave request not submitting**

**Solutions:**
- Check all required fields filled
- Ensure dates are valid (not in past)
- Verify reason is provided
- Try creating as draft first

### **Problem: Manager cannot see requests**

**Solutions:**
- Check user has Manager/HR role
- Verify department assignment
- Only see own department (unless admin)
- Check request status filter
