  📸 INSTALL FACE RECOGNITION LIBRARIES
═══════════════════════════════════════════════════════════════

To make face enrollment and login work, run these commands:

STEP 1: Install DeepFace (Face Recognition Engine)
───────────────────────────────────────────────────
pip install deepface

STEP 2: Install OpenCV (Image Processing)
──────────────────────────────────────────
pip install opencv-python

STEP 3: Install TensorFlow (AI Backend)
────────────────────────────────────────
pip install tensorflow

OR Install All At Once:
───────────────────────
pip install deepface opencv-python tensorflow


AFTER INSTALLATION:
═══════════════════
1. Restart your Django server
2. Go back to enrollment page
3. Try capturing face again
4. It will work!


WHAT THESE DO:
══════════════
✓ DeepFace - World-class face recognition (99.38% accuracy)
✓ OpenCV - Image processing and camera access
✓ TensorFlow - Deep learning AI for face matching


SYSTEM REQUIREMENTS:
════════════════════
- Python 3.8+
- 4GB RAM minimum (8GB recommended)
- Webcam (720p minimum, 1080p recommended)
- Internet connection (first run downloads AI models ~100MB)


INSTALLATION TIME:
══════════════════
- About 2-5 minutes depending on internet speed
- First face enrollment may take 10-15 seconds (downloads AI models)
- After that, enrollment takes 3-5 seconds per face


TROUBLESHOOTING:
════════════════
If installation fails:
1. Try: pip install --upgrade pip
2. Then retry the installations
3. Or install one by one instead of all at once

If you get "No module named 'tensorflow'":
- On Windows: pip install tensorflow
- The system will still work, just slower

═══════════════════════════════════════════════════════════════
Run: pip install deepface opencv-python tensorflow
Then restart server and try again!
