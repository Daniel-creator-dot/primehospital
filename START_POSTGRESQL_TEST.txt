                        🚀 QUICK START (15 MINUTES)
══════════════════════════════════════════════════════════════════════════════

Follow these steps in order:


STEP 1: Install PostgreSQL
══════════════════════════════════════════════════════════════════════════════

Run this command:
  install_postgresql.bat

This will:
  ✓ Open the PostgreSQL download page
  ✓ Guide you through installation
  ✓ Help you configure PostgreSQL

IMPORTANT: During installation:
  - Set a password for 'postgres' user (e.g., "admin123")
  - Remember this password!
  - Use port 5432 (default)
  - Install all components


STEP 2: Setup HMS Database
══════════════════════════════════════════════════════════════════════════════

After PostgreSQL installation completes, run:
  setup_postgresql_database.bat

This will:
  ✓ Create database: hms_test
  ✓ Create user: hms_user
  ✓ Set password: hms_password_123
  ✓ Grant all necessary permissions

You'll need to enter the 'postgres' password you set in Step 1


STEP 3: Install Python PostgreSQL Driver
══════════════════════════════════════════════════════════════════════════════

Run:
  install_requirements.bat

This installs: psycopg2-binary


STEP 4: Update .env File
══════════════════════════════════════════════════════════════════════════════

Create or edit .env file in your project root and add:

DATABASE_URL=postgresql://hms_user:hms_password_123@localhost:5432/hms_test
DATABASE_SSL_MODE=prefer
DATABASE_CONN_MAX_AGE=600
DATABASE_CONN_HEALTH_CHECKS=True


STEP 5: Test Connection
══════════════════════════════════════════════════════════════════════════════

Run:
  python test_db_connection.py

Expected output:
  [SUCCESS] Database connection SUCCESSFUL!
  PostgreSQL Information:
    Version: PostgreSQL 16.x
    Database: hms_test
    Server Address: 127.0.0.1
    Port: 5432


STEP 6: Run Migrations
══════════════════════════════════════════════════════════════════════════════

Run:
  python manage.py migrate

This creates all tables in PostgreSQL


STEP 7: Create Superuser
══════════════════════════════════════════════════════════════════════════════

Run:
  python manage.py createsuperuser

Enter username, email, and password


STEP 8: Start Server
══════════════════════════════════════════════════════════════════════════════

Run:
  python manage.py runserver

Visit: http://127.0.0.1:8000

══════════════════════════════════════════════════════════════════════════════
                        📊 COMPARE: BEFORE vs AFTER
══════════════════════════════════════════════════════════════════════════════

BEFORE (SQLite):
  DATABASE_URL=sqlite:///db.sqlite3
  - Simple file-based database
  - Good for development
  - Limited features
  - Not production-ready

AFTER (PostgreSQL):
  DATABASE_URL=postgresql://hms_user:hms_password_123@localhost:5432/hms_test
  - Production-grade database
  - Better performance
  - Advanced features
  - Handles concurrent users
  - Ready to scale to remote server


══════════════════════════════════════════════════════════════════════════════
                        🔧 USEFUL POSTGRESQL COMMANDS
══════════════════════════════════════════════════════════════════════════════

Access PostgreSQL command line:
  cd "C:\Program Files\PostgreSQL\16\bin"
  psql.exe -U postgres

Common commands (in psql):
  \l                           - List databases
  \c hms_test                  - Connect to hms_test database
  \dt                          - List tables
  \d hospital_patient          - Describe patient table
  \du                          - List users
  \q                           - Quit

Check database size:
  SELECT pg_size_pretty(pg_database_size('hms_test'));

Check patient count:
  SELECT COUNT(*) FROM hospital_patient;


══════════════════════════════════════════════════════════════════════════════
                        🔍 TROUBLESHOOTING
══════════════════════════════════════════════════════════════════════════════

PostgreSQL service not starting?
  - Open Services (services.msc)
  - Find "postgresql-x64-16"
  - Right-click → Start

Can't connect?
  - Check if PostgreSQL service is running
  - Verify password in .env matches what you set
  - Try: python test_db_connection.py

psycopg2 won't install?
  - Try: pip install psycopg2-binary
  - Or: pip install psycopg2

Database already exists error?
  - In psql: DROP DATABASE hms_test;
  - Then run setup_postgresql_database.bat again
