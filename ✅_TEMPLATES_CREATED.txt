======================================================================
                        🎯 HOW TO USE
======================================================================

FOR ADMINISTRATORS:

1. Go to: http://127.0.0.1:8000/hms/procurement/admin/pending/

2. You'll see table with:
   - Request numbers
   - Requesters
   - Items
   - Amounts
   - "Review" button

3. Click "Review" on any request

4. You'll see:
   - Full request details
   - All items listed
   - Total amount
   - Justification

5. Decision:
   - Click "Approve Request" → Request goes to accounts
   - Click "Reject Request" → Enter reason, reject

6. Done!


FOR ACCOUNTS:

1. Go to: http://127.0.0.1:8000/hms/procurement/accounts/pending/

2. See admin-approved requests

3. Click "Review"

4. Check details

5. Click "Approve Request"

6. ✨ MAGIC:
   - System creates Accounts Payable
   - System creates Expense Entry
   - System creates Payment Voucher
   - Success message shows entry numbers!

7. Done!
