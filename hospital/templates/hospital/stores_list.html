Transfer guidelines
  • Transfer out: Use “Transfer out” on a store card to move items from that store to another. You’ll pick items and quantities from that store’s real inventory; only available stock can be transferred.
  • Receive transfer: Use “Receive transfer” to create a transfer to this store from another (e.g. Main Store → Pharmacy). You still choose the source store and items on the next screen.
  • Workflow: (1) Create transfer → (2) Approve (if required) → (3) Complete when items are received. Only when you complete the transfer are quantities deducted from the source and added to the destination.
  • Quantities: Items and quantities are tied to the source store’s inventory. When the transfer is completed, the system updates the correct stock in both stores; if the destination doesn’t have the item, it is created automatically.

Need to start a transfer? Use New Transfer above, or the Transfer out / Receive buttons on each store card.
