How to create a transfer
  1. Choose stores: Select From store (where the stock is) and To store (where it should go). They must be different.
  2. Pick items: After the source store loads, select items from the list. Each item shows available quantity — you can only transfer up to that amount.
  3. Set quantities: Enter how many units to transfer for each item. Quantities are checked against real inventory; you cannot exceed available stock.
  4. Submit: Click Create Transfer. The transfer is created as Pending. An approver can then approve it; when someone completes the transfer, stock is deducted from the source store and added to the destination store.

Items and quantities are linked to the source store’s inventory, so completion updates the correct stock. If the destination store does not have the item, it is created automatically when the transfer is completed.
