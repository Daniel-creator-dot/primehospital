================================================================================
                      TROUBLESHOOTING
================================================================================

Problem: "Module not found"
Fix: pip install -r requirements.txt

Problem: "Table already exists"
Fix: Add --skip-drop flag (already in the commands above)

Problem: "No such file"
Fix: Check SQL files exist in C:\Users\user\Videos\DS\

Problem: Still can't see data
Fix: Run python manage.py validate_import --detailed


================================================================================

       patients = cursor.fetchall()

3. Generate Django model:
   python manage.py inspectdb patient_data > patient_model.py

4. View in admin (after creating model and registering)
