══════════════════════════════════════════════════════════════════════════════
                        IMPORTANT NOTES
══════════════════════════════════════════════════════════════════════════════

1. LEGACY DATA IS READ-ONLY
   - You can VIEW and SEARCH
   - You CANNOT add or delete (preserves historical data)
   - This is intentional for data integrity

2. TWO PATIENT SYSTEMS
   - "Patients" = New Django system (Marilyn Ayisi, etc.)
   - "Legacy Patients" = Imported old data (35,019 records)
   - Both are useful for different purposes

3. DIFFERENT MRN FORMATS
   - New patients: PMC2025000028 (Django generated)
   - Legacy patients: PID numbers like 1, 2, 3, 2021, etc.

4. SEARCH WORKS IN BOTH
   - Search new patients in "Patients" section
   - Search imported patients in "Legacy Patients" section
