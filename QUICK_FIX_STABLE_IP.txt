========================================
QUICK FIX: Make Your IP Address Stable
========================================

PROBLEM: Your IP address changes every day
SOLUTION: Set a static IP address

========================================
OPTION 1: Automatic (Easiest)
========================================

1. Right-click: SET_STATIC_IP.bat
2. Select: "Run as administrator"
3. Follow the prompts
4. Done! Your IP will never change again

========================================
OPTION 2: Manual Windows Settings
========================================

1. Press Win + I
2. Go to: Network & Internet → Properties
3. Click: "Edit" next to IP assignment
4. Select: "Manual"
5. Enter your current IP, subnet, gateway, DNS
6. Save

========================================
OPTION 3: Router Configuration (Best)
========================================

1. Open router admin: http://192.168.0.1
2. Find: DHCP Reservations / Static IP Assignment
3. Add your computer's MAC address
4. Assign IP: 192.168.0.100 (or any available)
5. Save and restart computer

========================================
CURRENT STATUS
========================================

✅ Your HMS already accepts dynamic IPs!
   - Works with: 192.168.0.*, 192.168.1.*, 192.168.2.*
   - Works with: 10.*, 172.16-31.*
   
✅ But static IP is still better for:
   - Consistent access URLs
   - Easier firewall setup
   - Better device management

========================================
AFTER SETTING STATIC IP
========================================

1. Run: AUTO_UPDATE_IP.bat (updates firewall script)
2. Access HMS: http://YOUR_STATIC_IP:8000
3. Share URL with other devices on network
