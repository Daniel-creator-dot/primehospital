## Format Structure

**PCMC-DEPT-PROF-NUMBER**

1. **PCMC** - PrimeCare Medical Center (Institution prefix)
2. **DEPT** - Department code (3 uppercase letters)
3. **PROF** - Profession code (3 uppercase letters)
4. **NUMBER** - Sequential number (4 digits with leading zeros)

## Department Codes

- Accounts → `ACC`
- Nursing → `NUR`
- Pharmacy → `PHA`
- Laboratory → `LAB`
- X-ray → `XRX`
- IT Support → `ITX`
