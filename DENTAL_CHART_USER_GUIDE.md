# Modern Dental Chart - User Guide

## Quick Start Guide

### Accessing the Dental Chart
1. Navigate to **Specialists** → **Dental Consultation**
2. Select a patient from the active encounters list
3. The interactive dental chart will load automatically

## Features Overview

### 🦷 Interactive Tooth Selection
- **Click any tooth** to select it
- Selected teeth glow with a purple hue and pulse
- **Hover** over teeth for instant visual feedback
- View tooth information in the info panel below the chart

### 🎨 Condition Color Coding

The chart uses an intuitive color system:

| Condition | Color | Visual Indicator |
|-----------|-------|------------------|
| **Healthy** | White with green border | Clean appearance |
| **Carious (Cavity)** | Light red with red border | Indicates decay |
| **Filled** | Light blue with blue border | Shows restoration |
| **Crown** | Light yellow with amber border | Gold-like appearance |
| **Bridge** | Light purple with purple border | Connected treatment |
| **Implant** | Light green with emerald border | Permanent replacement |
| **Root Canal** | Peach with orange border | Endodontic treatment |
| **Extraction Needed** | Red with **blinking** border | ⚠️ Urgent attention |
| **Missing** | Gray with dashed border | Faded appearance |
| **Erupting** | Light cyan with dashed border | Developing tooth |

### 📍 Tooth Numbering System

The chart displays **two numbering systems**:
- **Sequential Number** (1-32): Top number, larger font
- **FDI Number**: Bottom number, smaller gray font

#### Quadrant Layout:
```
        Upper Right  |  Upper Left
        (18-11)      |  (21-28)
        Teeth 1-8    |  Teeth 9-16
    ─────────────────────────────────
        Lower Right  |  Lower Left  
        (48-41)      |  (31-38)
        Teeth 25-32  |  Teeth 17-24
```

### 🎯 How to Mark Tooth Conditions

1. **Select a tooth** by clicking it
2. Choose a **condition type** from the dropdown
3. Condition is **automatically saved**
4. Tooth color updates **instantly**
5. Success notification appears in top-right corner

### 📝 Adding Procedures

1. Click **"Add Procedure"** button
2. Fill in the procedure details:
   - Procedure Code & Name
   - Procedure Type (Extraction, Filling, Crown, etc.)
   - Teeth affected (comma-separated, e.g., "11,12,13")
   - Quantity and Fee
   - Notes
3. Click **"Save Procedure"**
4. Procedure appears in the list below

### 🔄 Managing Selections

- **Clear Selection**: Click the red "Clear Selection" button
- **Multiple Teeth**: Click multiple teeth before adding a procedure
- **Selected teeth** are automatically added to the procedure form

### 📊 Procedure Management

Use the tabs to filter procedures:
- **All Procedures**: View complete history
- **Planned**: Future treatments
- **In Progress**: Currently ongoing
- **Completed**: Finished treatments

### 💡 Tips & Best Practices

#### For Dentists:
- ✅ Mark conditions as you examine each tooth
- ✅ Use the FDI numbers for professional communication
- ✅ Document all procedures with detailed notes
- ✅ Update tooth conditions after completing procedures
- ✅ Review the legend if unsure about color meanings

#### For Admin Staff:
- ✅ Sequential numbers (1-32) are easier for data entry
- ✅ Verify selected teeth before adding procedures
- ✅ Use the procedure catalog for consistent pricing
- ✅ Check procedure status regularly

### 🖱️ Keyboard & Navigation

- **Tab**: Navigate between controls
- **Enter**: Confirm selections
- **Escape**: Close modals
- **Mouse Wheel**: Scroll through procedure list

### 📱 Mobile Support

The chart is **fully responsive**:
- Scales automatically on tablets
- Touch-friendly tap targets
- Optimized legend layout for small screens
- Swipe to scroll procedures

### 🎨 Visual Indicators

#### Tooth States:
- **Normal**: Standard appearance, no condition
- **Hovered**: Slightly enlarged, shows interactivity
- **Selected**: Purple glow, pulsing animation
- **Missing**: Faded with dashed border
- **Urgent**: Blinking animation (extraction needed)

#### Animations:
- ✨ **Pulse**: Selected teeth pulse gently
- ✨ **Blink**: Urgent conditions blink slowly
- ✨ **Slide**: Info panel slides in smoothly
- ✨ **Rotate**: Background gradient rotates subtly

### 🔧 Troubleshooting

#### Chart Not Loading?
- Ensure patient has an active encounter
- Check browser console for errors
- Refresh the page (F5)

#### Can't Select Teeth?
- Make sure the chart is fully loaded
- Clear previous selection first
- Try clicking the center of the tooth

#### Conditions Not Saving?
- Check your internet connection
- Verify you have the necessary permissions
- Look for error notifications in top-right

#### Colors Not Showing?
- The default state is "healthy" (white/green)
- Select a condition from the dropdown
- Click a tooth to apply the condition

### 📖 Terminology

- **FDI Notation**: International standard tooth numbering (11-48)
- **Sequential**: Simple 1-32 numbering system
- **Quadrant**: One-quarter of the mouth (4 total)
- **Maxilla**: Upper jaw
- **Mandible**: Lower jaw
- **Arch**: The curved arrangement of teeth

### 🎓 Training Resources

For new users:
1. Start with the **legend** at the bottom of the chart
2. Practice selecting teeth without saving
3. Learn the **FDI numbering** gradually
4. Use **sequential numbers** initially if needed
5. Reference the **quadrant labels** for orientation
