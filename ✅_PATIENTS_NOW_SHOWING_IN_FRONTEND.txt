                        TROUBLESHOOTING
══════════════════════════════════════════════════════════════════════════════

"Still showing 48 patients"
  → Make sure you selected "Imported Legacy" or "All" from dropdown
  → Restart server: Ctrl+C then python manage.py runserver
  → Hard refresh browser: Ctrl+Shift+R

"Don't see Source dropdown"
  → Server not restarted
  → Restart: python manage.py runserver
  → Clear cache: Ctrl+Shift+R

"No patients showing"
  → Verify data: python verify_legacy_patients.py
  → Should show 35,019 records
  → If not: python import_patient_final.py
