======================================================================
                        🎯 HOW IT WORKS
======================================================================

SMART DETECTION:
  1. Dashboard checks user permissions
  2. If has admin permission → Checks for 'submitted' requests
  3. If has accounts permission → Checks for 'admin_approved' requests
  4. Counts pending items
  5. Shows alert if count > 0

PERMISSION-BASED:
  - Regular users: No alerts
  - Administrators: See admin approval alerts
  - Accounts staff: See accounts approval alerts
  - Superusers: See both

REAL-TIME COUNTS:
  - Updates every page load
  - Shows exact number
  - No caching
